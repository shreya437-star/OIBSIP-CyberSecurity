# OIBSIP-CyberSecurity
Cyber Security Internship Tasks - Oasis Infobyte (December P1)
## Task 1: Basic Network Scanning with Nmap

### Objective
To perform a basic network scan using Nmap to identify open ports and services.

### Command Used
nmap localhost

### Description
The scan was performed on the local system (localhost) to check which TCP ports are open.
Most ports were found closed, indicating a secure system.

### Conclusion
This task helped in understanding how Nmap can be used for basic network reconnaissance
and identifying running services on a system.
